## React Installation

Install the [AnyPay SDK](https://github.com/0xsequence/anypay/tree/master).

<CodeGroup>
```shell pnpm
pnpm i @0xsequence/anypay-sdk/widget
```
```shell npm
npm install --save @0xsequence/anypay-sdk/widget
```
```shell yarn
yarn add @0xsequence/anypay-sdk/widget
```
```shell bun
bun add @0xsequence/anypay-sdk/widget
```
</CodeGroup>

### Import Widget

```typescript
import { AnyPayWidget } from '@0xsequence/anypay-sdk/widget'

export const App = () => {
  return (
    <AnyPayWidget
      renderInline={true}
      theme="auto"
      walletOptions={[]}
    />
  )
}
```

## Script Import

While we recommend to utilize the SDK as a React component, you can alternatively import via a script in your `index.html`.

```typescript
<!DOCTYPE html>
<html>
<head>
    <title>AnyPay Widget Demo</title>
    <script src="https://anypay.sequence-demos.xyz/js/anypay.min.js"></script>
</head>
<body>
    <div id="anypay"></div>
    
    <script>
        AnyPayWidget.render(document.getElementById('anypay'), {
            renderInline: true,
            theme: 'auto',
            walletOptions: []
        });
    </script>
</body>
</html>
```
