You can try out the flow and demo of Trails on our sandbox environment [here](https://demo.trails.build/widget). 

To install on your own application:
<Steps>
  <Step title="Install the SDK">
    <CodeGroup>
    ```shell pnpm
    pnpm i 0xtrails
    ```
    ```shell npm
    npm install --save 0xtrails
    ```
    ```shell yarn
    yarn add 0xtrails
    ```
    ```shell bun
    bun add 0xtrails
    ```
    </CodeGroup>
  </Step>
  
  <Step title="Import and Use the Widget">
    Add the widget to your React application, for example send 1 USDC cross-chain to a recipient on Base.

    ```typescript
    import { TrailsWidget } from '0xtrails/widget'

    export const App = () => {
      return (
        <TrailsWidget
          toAddress="0x97c4A952b46bEcaD0663f76357d3776ba11566E1"
          toAmount="1" 
          toChainId={8453}
          toToken="USDC"
        />
      )
    }
    ```
  </Step>
</Steps>


## Script Import

While we recommend using the React component, you can also embed via a script tag in a non-React site.

```js
<div id="trails"></div>
<script src="/js/trails.min.js"></script>
<script>
  TrailsWidget.render(document.getElementById('trails'), {
    toAddress: '0x...',
    toAmount: '0.1',
    toChainId: 8453,
    toToken: '0xUSDC...'
  })
</script>
```
